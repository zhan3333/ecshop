# 声明

此版本为适配php5.6版本的ecshop代码。ecshop版本为2.7.3。

# 实例项目地址 (版本老旧，已停止访问)

- 商城： http://ecshop.grianchan.com
- 管理端： http://ecshop.grianchan.com/admin
    - 登陆账号： `zhan/zhan1234`

# 介绍

由于官网提供的 ecshop 源码老旧，需要在 php5.3 及以下版本中运行, 所以在这个项目中，修改了一些php的高版本兼容性问题，让项目能够运行在php5.6环境中。

# 运行环境

- php/5.6
- mysql/5.6
- Tengine/2.1.1
- Linux/CentOS 7.2
